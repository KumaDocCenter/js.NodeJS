# Node.js

## 基本信息

**WebSite**:  https://nodejs.org/zh-cn/
**GitHub**: https://github.com/nodejs/node
**维基百科**：https://en.wikipedia.org/wiki/Node.js

----------------------------------------------------------------------
Node.js是基于Chrome的V8 JavaScript引擎构建的JavaScript运行时。

Node.js的是一个开源，跨平台 的JavaScript 运行时环境，执行浏览器之外的JavaScript代码。

Node.js 不是一个 JavaScript 框架，不同于CakePHP、Django、Rails。
Node.js 更不是浏览器端的库，不能与 jQuery、ExtJS 相提并论。
Node.js 是一个让 JavaScript 运行在服务端的开发平台，它让 JavaScript 成为与PHP、Python、Perl、Ruby 等服务端语言平起平坐的脚本语言。
